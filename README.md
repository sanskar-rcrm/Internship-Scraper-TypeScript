# Internship Scraper (TypeScript)

This project scrapes internship listings from **AICTE**, **Skill India**, and **IIRS (ISRO)** websites and generates a unified `jobs.json` file with clean structured data.

---

## Features
- Built fully in **TypeScript**
- Scrapes and normalizes internships from **multiple government sources**
- Combines and assigns unique `job_id` to every internship
- Outputs a clean JSON dataset ready for search engines / ML / dashboards

---

## Installation
```bash
npm install
```

## Running
